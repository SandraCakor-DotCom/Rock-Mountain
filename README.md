# Rock Mountain – Live Music Website

A modern landing page for a fictional live music band called **Rock Mountain**.  
The website presents upcoming shows, music content and social media links.

## Features

- Hero section with band branding
- Navigation menu (Home, Locations, About, Contact)
- Sections for News, Album and Store
- Event information with ticket purchase button
- Video / media section
- Social media integration
- Newsletter subscription form
- Responsive design for mobile devices

## Technologies

- HTML5
- CSS3
- Google Fonts
